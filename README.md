# SwiftShare

基于Vue3开发的霉霉视频分享平台，为Swiftie们提供视频分享服务。

## 技术栈

- Vue 3.2.13
- Element Plus 2.3.7
- Vue Router 4.2.4
- Vuex 4.1.0
- Axios 1.4.0
- Video.js 8.5.2
